# ヨソノナ
ヨソノナとは、漢字から人名・名字を推測するウェブアップです。

## このプロジェクトについて

当プロジェクトは2023/2度チュラロンコーン大学で行われる「Applied Japanese Linguistics」（2223332）のプロジェクトです。<br>
ご連絡、ご意見などは[こちらのメール](mailto:contact@yuugenp.xyz)までお願いいたします。
